# Product Overview

EPM Note Engine（Code Name: ValueCruise）は、経営管理SaaS「EPM」の認知獲得・リード創出のため、高品質なNote記事を半自動生成し、投稿負荷を極限まで下げるマーケティング自動化ツール。

## Core Capabilities

1. **SEO & Quality**: 競合分析と自己推敲（Self-Correction）により、検索上位を狙える品質を担保
2. **Asset Management**: ユーザーの暗黙知（失敗談・画像・音声）を構造化データとして資産化
3. **Full Automation**: 記事作成だけでなく、SNS投稿文の生成からNoteへの下書き保存までを自動化

## Target Use Cases

- EPMプロダクトの認知獲得のためのNote記事の継続的な投稿
- SEOを意識した高品質コンテンツの効率的な生成
- マーケティング担当者の記事作成負荷の大幅削減

## Value Proposition

**"Autonomous Marketing RAG with Last-Mile Automation"**

- 競合分析 + 社内知見のハイブリッドリサーチで差別化された記事を生成
- LangGraphによる品質スコアリング・自動修正ループで一定品質を担保
- Playwrightによる「最後の1マイル」（Note.comへの下書き保存）まで自動化

## AI活用の位置づけ

- AIは**執筆を加速する道具**であり、最終判断は人間が行う
- Claude 3.5 Sonnet（メイン執筆）+ GPT-4o（検索・マルチモーダル）の使い分け
- 生成された記事は必ず人間がレビュー・編集してから公開

## 運用想定

- **利用者**: Product Owner（Tomoyuki氏）
- **投稿頻度**: 週2本を目標
- **ワークフロー**: 記事候補選定 → AI生成 → 人間レビュー → Note投稿
- **品質基準**: EPMの専門性を示しつつ、初心者にも分かりやすい記事
